Comment utiliser le CSS

Le CSS sert à donner du style au site web, rajouter une touche d'extravagance et de personalité a celui ci

Relier le fichier CSS avec le fichier HTML afin que les modifications apparaissent sur votre site

Pour ce faire il faut réaliser un code qui devra etre situé dans la balise "<head>" ce code est le code: <link rel="stylesheet" type="text/css" href="style.css">

Quelque astuce du CSS:

On peut changer la couleur du fond en utilisant la balise: body{

background-color: la couleur de vitre choix;
}

ou encore changer la couleur de l'écriture de vos paragraphes ect ect: h1{ color: la couleur de votre choix; }
